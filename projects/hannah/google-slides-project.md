## What is Google Slides?

Google Slides is a free, web-based presentation program that competes with Microsoft PowerPoint. Part of Google Workspace, it allows users to create, edit, and share presentations entirely in the cloud. Available on all devices via web browser or mobile apps (Android/iOS).

**Key features:** Real-time collaboration, automatic cloud saving, version history, offline editing, cross-platform accessibility.

**Supported file types:** PPT, PPTX, ODP, JPG, PNG, POT, TXT, SVG, PDF

**Sources:** [How-To Geek](https://www.howtogeek.com/428140/the-beginners-guide-to-google-slides/) (Jan 2025)

## Getting Started

### Creating a Presentation
1. Go to [slides.google.com](https://slides.google.com) and sign in with a Google account
2. Click the + button or choose a template to start
3. Name your file at the top-left
4. Adjust page setup under **File > Page Setup** for slide dimensions

### Adding Content
- **Insert menu:** Images, video, audio, shapes, tables, charts, diagrams, word art
- **Add slide:** + button (top-left) or right-click navigation panel > duplicate
- **Themes:** Choose from built-in themes or customize backgrounds per slide
- **Fonts:** Customizable font styling, colours, and sizes

### Key Shortcuts
- `Ctrl+D` — Duplicate slide
- `Ctrl+Shift+C` — Copy formatting
- `Ctrl+Shift+V` — Paste formatting
- `Ctrl+K` — Insert link

## Advanced Features

### Collaboration
- **Real-time editing:** Multiple users can edit simultaneously; see their cursors
- **Comments:** Add, reply, assign with @mention (sends email notification)
- **Chat:** Built-in chat window with active collaborators
- **Email collaborators:** File > Email collaborators to send messages
- **Version history:** File > Version history > See version history — restore previous versions or copy from them

### Multimedia
- **Audio:** Embed audio files (record separately with tools like ChromeMP3 Recorder)
- **Video:** Insert YouTube videos or Drive video files directly
- **Animations & Transitions:** Object animations, slide transitions, custom timing

### Integrations
- **Charts:** Insert linked Google Sheets charts
- **Diagrams:** Built-in diagram templates
- **Google Workspace integration:** Link with Docs, Sheets for data import

### Offline Mode
- Enable offline editing in Google Drive settings
- Changes sync when reconnected

## Cybersecurity Advisory — Google Slides

When using Google Slides, take note of the following security considerations:

**Sharing & Access Control**
- Only share with people you trust. Review sharing permissions regularly (View / Comment / Edit).
- Do not set a sensitive presentation to "Anyone with the link can view" unless necessary.
- Be aware that collaborators with Edit access can change permissions, add others, or delete content.
- For sensitive content, use "Restricted" sharing and add specific people only.

**Phishing via Shared Documents**
- Attackers may send fake Google Slides share invitations via email to steal credentials.
- Only accept sharing invitations you were expecting. Verify the sender.
- Check the URL: legitimate Google Slides URLs start with `docs.google.com/presentation/`.

**Third-Party Add-ons**
- Be cautious when installing Slides add-ons from third-party developers.
- Review the permissions they request before authorising.

**Account Security**
- Enable 2-factor authentication on your Google account.
- Use a strong, unique password.
- Check Recent Security Events in your Google Account settings monthly.

**Data Leakage via Version History**
- Deleted content may still exist in version history. Before sharing a presentation externally, review and clear sensitive versions.
