## 🎨 Design Styles / Themes

### 1. Y2K / Retro-Futurism (seems like your vibe, Hannah)
**What it is:** Chrome textures, iridescent gradients, see-through/glassy UI panels, pixel fonts, low-res textures. The early internet aesthetic — MySpace, iMac G3 era. Big in 2026 right now.
**Keywords:** chrome, iridescent, transparent plastic, pixel artifacts, body-tech fusion, bubbly letters, cotton-candy colors, Winamp skins energy.
**Color palette:** Crayola-bright pastels (pink, lime, cyan), metallics, dark charcoal backgrounds.
**Where it works:** Personal profile, portfolio, music/artist sites.

### 2. Hacker / Terminal Theme
**What it is:** Green-on-black CRT monitor look, monospace fonts, terminal emulator interfaces, Matrix rain backgrounds, command-line navigation.
**Cool resources:**
- `hacker.css` — CSS framework, one stylesheet = instant hacker look
- `jQuery.terminal` — JS library for fully interactive web terminals
- GitHub has 28+ terminal-style portfolio repos ready to fork
**Color palette:** `#00ff00` on `#000000`, amber/orange for warm variant.
**Where it works:** Developer portfolios, cyberpunk brands, 404 pages.

### 3. Minimalist / Clean + Bold Typography
**What it is:** Expansive white space, asymmetrical layouts, big bold headlines, subtle micro-animations, dark mode toggle.
**Key trends 2026:** Intentional asymmetry, expansive white space, scroll-triggered reveals, subtle motion.
**Where it works:** Creative portfolios, writer/designer sites.

### 4. Neocities / Indie Web
**What it is:** Raw HTML/CSS, handmade feel, blinking GIFs, guestbooks, hit counters, MIDI backgrounds. The anti-minimalist web. Pure expression.
**Where it works:** Personal zines, art projects, anything where you want maximum personality.

## ✨ Interactive Features (Make Your Site Pop)

| Feature | What It Does | Good For |
|---------|-------------|----------|
| **Scroll-triggered animations** | Content fades/slides in as you scroll | Any long page |
| **Cursor effects** | Trails, particles, glow following mouse | Flexing design skills |
| **Parallax scrolling** | Background moves slower than foreground | Immersive storytelling |
| **3D tilt cards** | Cards rotate as mouse moves | Showcasing projects |
| **Dark/light toggle** | Switch between themes | UX win, every site needs this |
| **Typewriter/typing effect** | Text types itself | Hero section, bio |
| **Confetti on click** | Explosion of color | Celebrations, portfolio milestones |
| **Hover sound effects** | Play sounds on mouseover | Meme energy |
| **Floating/sticky note** | Persistent draggable note | Personal touch |
| **Music player** | Background music toggle | Setting the vibe |

## 🔥 Unique Personal Website Ideas

### Idea A: "The Digital Garden"
Not a portfolio — a growing collection of thoughts, projects, recipes, playlists, photos. Meant to be messy, alive, updated regularly. Like a Neocities but yours.
**Sections:** `/now` (what you're up to), `/uses` (tools you use), `/books`, `/music`, `/art`.

### Idea B: Terminal-Based Portfolio
Type commands to navigate. `ls` lists projects, `cat about.txt` shows bio, `cd projects` enters project section. Full interactive terminal experience.
**Stack:** jQuery Terminal or a simple Vanilla JS terminal.

### Idea C: Y2K Cyber Sanctuary
Pink/cyan gradients, butterfly GIFs, glitter text, "Under Construction" banners (ironic), guestbook, MySpace-style top 8, Winamp skin aesthetic. Pure early 2000s chaos.
**Perfect for:** A fun personal homepage separate from your professional PG.

### Idea D: One-Page Infinite Scroll
Everything on one long, beautifully designed page. Smooth scroll navigation, each section is a "chapter" of your life/story. Bold background colors change per section.
**Sections:** Hero → About → Skills → Projects → Blog → Contact.

### Idea E: Windows Desktop Clone
Website that looks like an actual Windows 95/98/Xp desktop. Icons you click to open "windows" (modals) for each section. Taskbar, start menu, wallpaper.
**Sections:** Each app = a section (Notepad = bio, Paint = gallery, Terminal = skills).

### Idea F: Zine / Magazine Grid
Heavy editorial layout. Big images, tight grids, article-style project descriptions. Like a digital magazine where you're the cover story.

## 📸 What Real People's Personal Sites Include

From browsing 100+ examples, the winning formula:

1. **Hero section** — Your name + one-liner tagline + main photo
2. **About me** — First person, real personality, hobbies included (guitar, badminton, sewing — these make you human 🏸🎸🧵)
3. **Projects / Work** — Quality over quantity (3 good ones > 10 mediocre)
4. **Skills** — Visual bars or tags
5. **Contact** — Social links + simple form
6. **Extra flair** — Guestbook, blog, playlist, guestbook, music player
7. **Footer** — "Built by Hannah with ❤️"

**Key stat:** Personal websites increase interview callbacks by **71%**.
**Key stat:** 70% of employers check personal websites during hiring.
