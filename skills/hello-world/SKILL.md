# Hello World Skill

**Description:** Mynyen's school study station — Pomodoro timer, assignment to-do list, and daily habit tracker. All in one self-contained HTML page.

## Files

- `study-station.html` — The Study Station dashboard

## How to Use

Open `study-station.html` in any browser. Everything runs locally in your browser — data saves to localStorage so it persists between sessions.

## Features

- **Pomodoro Timer** — 25 min focus / 5 min break, tracks sessions
- **Assignments** — To-do list with checkboxes, add/delete/clear completed
- **Habit Tracker** — Weekly checkboxes for each habit, add custom habits
- **Auto-save** — All data saves to your browser automatically

## Usage

```bash
# Open directly
xdg-open ~/.openclaw/workspace/skills/hello-world/study-station.html
```
